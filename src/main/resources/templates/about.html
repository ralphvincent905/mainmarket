Project Overview

MainMarket is a demo e-commerce platform built as part of the CPAN 228 Web Application Development course. The project demonstrates how modern web applications are built using Spring Boot, Thymeleaf, and Bootstrap.

Users can browse product listings, add new products through a form interface, and explore marketplace data stored in a relational database.

Technologies Used
  • Spring Boot
  • Thymeleaf Templates
  • Bootstrap 5
  • Java & JPA

This project demonstrates a basic microservice-based marketplace system with a focus on clean UI design, data management, and responsive web development.
