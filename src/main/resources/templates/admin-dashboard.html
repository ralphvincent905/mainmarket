📋 Role-Based Access Control

ADMIN: Full system access - can manage users, products, view analytics, and system settings. Can perform all operations in the system.
STAFF: Can manage products (create, edit, delete), view orders and customer information, but cannot manage users or access admin settings.
CUSTOMER: Can browse products, view their profile, make purchases, and track orders. Cannot access product management or admin features.
